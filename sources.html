Это файл sources.html репозитария cis1-docs. В нем содержится описание всех репозитариев системы cis1.

Репозитарии системы cis1

cis1-docs репозитарий с базовой документацией (этот файл находится в этом самом репозитарии)
cis1-core-bash репозитарий компоненты Ядро, написанном на bash

cis1-webui-native-front-vuejs часть компоненты WEBUI, написанной на С++, содержит frontend код
